# AI Interview Agent

An intelligent technical interviewer and candidate evaluation agent built with **FastAPI / Express, React, TypeScript, Tailwind CSS**, and **Google Gemini 3.6 Flash**.

The AI Interview Agent automates technical screening interviews by generating role-specific questions across escalating difficulty levels, capturing candidate answers, evaluating technical accuracy, scoring each question out of 10, and producing executive hiring reports.

---

## 🚀 Working Flow & Architecture

```
Start Project
      │
      ▼
Enter Job Role (e.g., Java Developer, Python Developer, AI Engineer)
      │
      ▼
Generate 5 Interview Questions (Easy ➔ Medium ➔ Medium ➔ Hard ➔ Scenario Based)
      │
      ▼
Candidate Answers Questions (Text or Speech Voice Recording)
      │
      ▼
AI Evaluation (Score out of 10, Strengths, Missing Elements, Model Answer)
      │
      ▼
Final Report Generation (Overall Score / 50, Status, Strengths, Weaknesses, Resources)
      │
      ▼
Save in SQLite Database & Export transcript.json
```

## ⚡ API Design & Endpoints

### 1. Generate 5 Questions
- **Endpoint**: `POST /api/generate`
- **Request Body**:
  ```json
  {
    "role": "Java Developer",
    "candidateName": "John Doe",
    "experience": "Mid-Level"
  }
  ```
- **Response**:
  ```json
  {
    "success": true,
    "sessionId": "int-1784732849-ab123",
    "session": {
      "questions": [
        { "id": 1, "question": "What is JVM?", "difficulty": "Easy", "focus": "Core Java" },
        { "id": 2, "question": "Difference between ArrayList and LinkedList?", "difficulty": "Medium", "focus": "Collections" },
        { "id": 3, "question": "Explain Multithreading and Synchronization.", "difficulty": "Medium", "focus": "Concurrency" },
        { "id": 4, "question": "What happens when two threads update the same object?", "difficulty": "Hard", "focus": "Memory Model" },
        { "id": 5, "question": "Design an Employee Management API.", "difficulty": "Scenario Based", "focus": "System Design" }
      ]
    }
  }
  ```

### 2. Submit & Evaluate Answer
- **Endpoint**: `POST /api/answer`
- **Request Body**:
  ```json
  {
    "sessionId": "int-1784732849-ab123",
    "questionNumber": 1,
    "question": "What is JVM?",
    "difficulty": "Easy",
    "candidateAnswer": "Java Virtual Machine loads .class bytecode files...",
    "role": "Java Developer"
  }
  ```
- **Response**:
  ```json
  {
    "success": true,
    "evaluation": {
      "score": 8,
      "feedback": "Good explanation of bytecode loading and JIT compiler.",
      "correctAnswer": "JVM is an abstract computing machine that provides a runtime environment...",
      "topicsToImprove": ["Bytecode verification steps"]
    }
  }
  ```

### 3. Generate Final Report
- **Endpoint**: `POST /api/result`
- **Response**:
  ```json
  {
    "success": true,
    "report": {
      "overallScore": 40,
      "maxScore": 50,
      "percentage": 80,
      "status": "Selected",
      "recommendation": "Suitable for Mid-Level Java Developer role.",
      "strengths": ["Core Java", "Collections", "Spring REST"],
      "weaknesses": ["Advanced Concurrency"],
      "learningResources": [...]
    }
  }
  ```

## ⚙️ Setup & Running

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Configure Gemini API Key**:
   Create a `.env` file or provide via AI Studio environment variables:
   ```env
   GEMINI_API_KEY="your_google_gemini_api_key"
   ```

3. **Start Development Server**:
   ```bash
   npm run dev
   ```
   Server will start on `http://0.0.0.0:3000`.

4. **Production Build**:
   ```bash
   npm run build
   npm run start
   ```

## ✨ Features Implemented
- **Role Selection**: Preset roles (Java, Python, AI Engineer, Full Stack, Frontend, Backend, Data Science, System Design, DevOps) + Custom roles.
- **5 Questions Generator**: Escalating difficulty (Easy, Medium, Medium, Hard, Scenario Based).
- **Interactive Answering**: Dual speech-to-text voice dictation or rich code/text input with countdown timers.
- **Per-Question Scoring**: Instant feedback with scores out of 10, strengths, missing points, and model answers.
- **Final Executive Report**: Confetti animations, hiring verdict, strengths/weaknesses breakdown, and curated learning path.
- **SQLite Database History**: Records persisted in `/data/interview.db.json` with history table and transcript viewer.
- **JSON Export**: Downloads full `transcript.json` for offline archiving and sharing.
